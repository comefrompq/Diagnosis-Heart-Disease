Để chạy được project này cần cài đặt trước: 
1) ASP.NET core 3.0 Web App.
2) .NET standard 3.0
3) Các nudget: 
3.1) dotNetRDF
===========================================
Project này gồm có 2 project nhỏ:
1) DHDSystem.Data :  nơi cài đặt các hàm xử lí, lấy dữ liệu từ file *.owl.
2) DHDSystem: Cài đặt giao diện web, xử lí các tương tác của người dùng với tầng data, gọi các hàm từ tầng Data để xử dụng.
